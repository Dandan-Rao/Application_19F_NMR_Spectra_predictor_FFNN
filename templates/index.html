How to use:
What is SMILES?

SMILES (Simplified Molecular Input Line Entry System) is a notation system for describing molecular structures using ASCII characters.

Tip: SMILES can be obtained from PubChem and ChemDraw.

Example SMILES:
  • PFBA: C(=O)(C(C(C(F)(F)F)(F)F)(F)F)O
  • PFBS: C(C(C(F)(F)S(=O)(=O)O)(F)F)(C(F)(F)F)(F)F
Requirements:
  • Must contain at least one fluorine atom (F)
  • Use valid SMILES syntax
  • Compound should be fluorinated
Processing time:

Analysis typically takes 50 ms to 10 seconds depending on compound complexity.
